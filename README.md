# Hotel-Website-PHP-

Royal Hotel – PHP Web Application (Faza I)
Përshkrimi i Projektit

Ky projekt ka për qëllim ndërtimin e një aplikacioni web për menaxhimin dhe prezantimin e një hoteli (“Royal Hotel”), i zhvilluar në PHP. Në Fazën I, aplikacioni fokusohet në strukturë reale të faqes, përdorim të koncepteve bazë të PHP-së dhe programimit të orientuar në objekte (OOP), pa përdorim të databazës.

Të dhënat janë të simuluara (dummy data), ndërsa aplikacioni është funksional dhe i navigueshëm.

Si të ekzekutohet projekti
Instaloni XAMPP / WAMP / Laragon.
Vendosni projektin në folderin htdocs.
Startoni Apache server.

Hapni browser dhe shkruani:

http://localhost/project/pages/index.php

1. Struktura e aplikacionit
Aplikacioni përmban më shumë se 4 faqe funksionale:
  -Home
  -About
  -Contact
  -Login
Përdoret include për:
  -header
  -navbar
  -footer
Projekti është i organizuar në folderë: pages, includes, classes, assets.

Shembull i përdorimit
  -Përdoruesi hap Home page
  -Navigon në About page
  -Mund të bëjë login
  -Sistemi ruan user-in në session
  -Përmbajtja ndryshon sipas rolit
